நாட்டிற்குள் உள்ள அனைத்து புலனாய்வு அமைப்புகளையும் மேற்பார்வையிடும் பொறுப்பு, CNI பாதுகாப்பு அமைச்சகத்தின் கீழ் செயல்படுகிறது. குலதுங்க 2019 இல் சிரேஷ்ட புலனாய்வு அதிகாரியாக தனது பங்கை ஏற்றார்.
